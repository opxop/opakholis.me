Sama halnya ketika memulai belajar bahasa _Programming_, ekspresi ini mungkin sangat familiar. Setiap kali belajar bahasa baru. Saya, mungkin juga Kamu pasti memulainya dengan membuat program sederhana yang tidak begitu berguna hanya dengan menampilkan kata **Hello World** pada _Console_ atau _Browser_. That's good, setidaknya itu berarti Kita mulai mempelajari sesuatu.

Dan tulisan ini—Hello World—merupakan tulisan pertama pada Blog ini dengan domain kepemilikan pribadi tentunya.

Sebagai pengenalan, Website / Blog ini dibangun dengan menggunakan beberapa teknologi yang memudahkan saya dalam proses pengembangan, diantaranya:

- Next.js — sebagai frontend framework
- Chakra UI — untuk styling (Component Library)
- Firebase — Hey, jangan sungakan untuk keluarkan pendapatmu di [Halaman ini](/advices)
- MDX — konten majamenen untuk orang ribet(?)
