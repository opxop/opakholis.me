Karena saya menjadikan [Firefox](https://www.mozilla.org) sebagai _daily browser_ pada setup kerja saya, jelas masalah ini sangat mengganggu sekali. Terutama ketika _project_ desain yang saya kerjakan sudah cukup komplek.

Oleh karena itu, dengan tulisan ini semoga bisa membantu memecahkan masalah ["scrolling Figma yang lambat pada Firefox (Linux)"](https://opakholis.me/blog/figma-mousewheel-issue) bagi kamu yang mengalami masalah sama seperti saya.

## Solve with extension

Solusi paling simple untuk mengatasi masalah ini bisa dengan meng-install [extension ini](https://addons.mozilla.org/firefox/addon/scroll-figma/).

![Figma addon](/static/images/figma-mousewheel-issue/scroll-figma-addon.png)

Jangan khawatir dengan rating + penggunanya yang terbilang sedikit, karena extension ini benar-benar mengatasi masalah yang kamu hadapi saat ini. Bahkan dibuat Open Source oleh pembuatnya. Untuk melihat sumber kodenya, kamu bisa berkunjung ke [Repository Github ini](https://github.com/800147/scroll-figma).

## If you a Greasemonkey user, try this one!

Bagi pengguna [Greasemonkey](https://addons.mozilla.org/firefox/addon/greasemonkey/), mungkin akan lebih berguna jika menggunakan script berikut. Saya asumsikan sudah paham bagaimana cara menggunakan extension satu ini.

```js:figma-mouse-wheel.user.js
const MULTIPLIER_X = 30;
const MULTIPLIER_Y = 10;

function speedUp(ev) {
  if (!ev.isTrusted) return true;

  const { clientX, clientY, screenX, screenY, shiftKey, ctrlKey, altKey } = ev;
  const newEv = new WheelEvent('wheel', {
    bubbles: true,
    cancelable: true,
    view: window,
    clientX,
    clientY,
    screenX,
    screenY,
    ctrlKey,
    shiftKey,
    altKey,
    deltaX: ev.deltaX * MULTIPLIER_X,
    deltaY: ev.deltaY * MULTIPLIER_Y
  });

  ev.preventDefault();
  ev.stopPropagation();
  ev.target.dispatchEvent(newEv);
  return false;
}

function addCanvasEvent() {
  const canvas = document.querySelector('.view > canvas');
  if (canvas != null) canvas.addEventListener('wheel', speedUp);
  else setTimeout(addCanvasEvent, 500);
}
addCanvasEvent();
```

## Poor man with complicated solution

> _"Kalo ada yang susah kenapa harus yang mudah"_

Hmm, begitulah yang terjadi pada penyelesaian masalah kali ini.

Perlu diketahui cara ini hanya mengatasi masalah Zoom in/out (**Ctrl+Mousewheel**). Jadi bagi yang memiliki masalah vertical (**Shift+Mousewheel**) dan horizontal scrolling (**Mousewheel**) sebaiknya menggunakan 2 cara di atas.

Oke. jadi gini kira-kira caranya, ya sebenernya gak terlalu susah juga sih.

Pertama buka `about:config`, tau caranya kan? tinggal ketik di _Address bar_, lalu masukan keyword berikut:

```bash
mousewheel.with_control.delta_multiplier_y
```

Setelahnya tinggal ubah _value_ dari konfigurasi tersebut, disini saya mengubahnya menjadi 1000. And that's it!
